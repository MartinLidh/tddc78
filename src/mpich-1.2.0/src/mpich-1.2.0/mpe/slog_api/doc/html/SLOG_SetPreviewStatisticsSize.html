SLOG_SetPreviewStatisticsSize

 Set/Override the default number of bins of the statistics array.

Synopsis


int SLOG_SetPreviewStatisticsSize(       SLOG_STREAM      *slog,
                                   const SLOG_uint32      stat_bin_size )

Unmodified Input parameter

stat_bin_size - the number of bin of the statistics array.

Modified Input parameter

slog - pointer to the SLOG_STREAM where interval records are stored.

Modified Output parameter

returned value - integer return status code.

Include file need

slog_pstat.h

Notes on SLOG return status code

There are 5 allowed SLOG returned status codes. They include SLOG_SUCCESS, SLOG_FAIL and SLOG_EOF. Their actual values are defined in slog.h

SLOG_SUCCESS is returned when no error is encountered.

SLOG_FAIL is returned when error is encounted.

SLOG_EOF is returned when End of file is encounted.

There are also SLOG_TRUE and SLOG_FALSE which are sued as boolean values, in particular they are used in boolean functions which are usually named as IsXXXXX(), AreXXXXX() and HasXXXXX().
